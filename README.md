# Crypto Investment Alert Concept

This repository contains sanitized sample materials for an AI-based crypto investment alert service concept.  
The project focuses on market data monitoring, alert logic, user risk notification, and service KPI design.  
No real API keys, private keys, account data, or live trading code are included.

본 저장소는 가상자산 투자 알림 서비스 기획을 위한 샘플 자료입니다. 실제 운영 코드나 민감정보는 포함하지 않았으며, 가격·거래량·이벤트 조건을 기반으로 사용자에게 리스크 알림을 제공하는 구조를 설명합니다.
